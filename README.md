# ferro-site

Source for [**ferro.abyo.net**](https://ferro.abyo.net) — the
Ferro ecosystem's public site. Built with [Zola](https://www.getzola.org/)
and deployed to GitHub Pages.

## Local preview

```bash
zola serve
# → http://127.0.0.1:1111
```

## Deploy

`main` auto-deploys via `.github/workflows/deploy.yml`.
